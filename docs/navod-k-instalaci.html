SubmiTheses

návod k lokálnímu spuštění

Požadavky
  1. Docker Engine & Docker Compose
  2. Git
  3. Bash (Linux / macOS / WSL na Windows)
Spuštění
  1. Naklonovat repozitář:
git clone https://github.com/krxgq/SubmiTheses.git
cd SubmiTheses
  1. Vytvořit konfigurační soubor prostředí:
cp .env.docker.example .env
  1. Sestavit a spustit kontejnery (schéma databáze se importuje automaticky):
docker compose up -d --build
  1. Vytvořit administrátorský účet:
bash seed-admin.sh
  1. Ověřit stav kontejnerů:
docker compose ps
Přístupové adresy
Aplikace http://localhost:3000
Backend API http://localhost:5000/api
Mailpit (e-maily) http://localhost:8025

Výchozí konfigurace v .env.docker.example je připravena pro lokální spuštění bez dalších úprav.
